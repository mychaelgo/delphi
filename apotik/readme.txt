Ini adalah program untuk olah data di apotik. Yaitu penjualan obat kepada 
pasien. Untuk menjalankan source code ini. Maka Anda harus memiliki komponen 
Zeos yang sudah terinstal di delphi Anda. Program ini saya buat dengan 
Borland Delphi 7.

Berikut petunjuk untuk menggunakan source code :
1. Buat database mysql misal : apotik.
2. dump file sql.sql
3. Cari file "libmysql.dll" yang terdapat pada "mysql\bin"
4. Copy-kan file tersebut ke "windows\system32"
5. lakukan perubahan pada zeosconn pada source code jika perlu.
6. jalankan.
